## 사용 가능한 macOS 시스템 사운드

| 사운드 | 추천 용도 |
|--------|----------|
| `Basso.aiff` | 에러, 경고 |

| `Bottle.aiff` | 가벼운 알림 |
| `Frog.aiff` | 재미있는 알림 |
| `Funk.aiff` | 주의 필요 |
| **`Glass.aiff`** | 현재 설정 ← |

| `Morse.aiff` | 메시지 |

| `Pop.aiff` | 팝업 |
| `Purr.aiff` | 부드러운 알림 |
| `Sosumi.aiff` | 클래식 Mac 사운드 |
| `Submarine.aiff` | 깊은 알림 |
| `Tink.aiff` | 가벼운 틱 |

## 전역 설정 여부

**네, 전역 설정입니다.**

| 파일 위치 | 범위 |
|-----------|------|
| `~/.claude/settings.json` | **전역** - 모든 프로젝트에 적용 ✅ |
| `프로젝트/.claude/settings.local.json` | 로컬 - 해당 프로젝트만 |

현재 Stop 훅은 `~/.claude/settings.json`에 있어서 **모든 프로젝트에서 종료음이 울립니다.**
