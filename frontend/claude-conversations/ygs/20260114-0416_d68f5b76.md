**추가된 기능:**

1. **유효성 검사 + 에러 표시**
   - 필드 입력 후 포커스를 잃으면(blur) 빈 값일 때 빨간 테두리 + 에러 메시지 표시

   - 에러 아이콘과 함께 한글 에러 메시지 표시

2. **자동 포맷팅**
   - 전화번호: `01012345678` → `010-1234-5678`
   - 생년: `1998` → `1998년`
   - 신장: `170` → `170cm`

3. **폼 레이아웃 개선**
   - 생년/신장을 한 줄에 2열로 배치 (데스크톱)
   - 필수 항목에 빨간 `*` 표시 추가

4. **검증 규칙**
   - 성명: 2자 이상
   - 연락처: 10-11자리 숫자
   - 생년: 1950~현재 사이 4자리
   - 신장: 100~250cm 사이
   - 직업/거주구: 필수 입력

5. **선택(select) 필드**
   - 요청대로 성별/학력/경로 드롭다운은 에러 메시지 없이 처리
