수정 완료했습니다.

**변경 사항:**
1. **자동 포맷팅**: 숫자만 입력하면 자동으로 `010-1234-5678` 형식으로 변환
2. **유효성 검사**: `01X-XXXX-XXXX` 패턴 체크
3. **에러 메시지**: 입력 후 포커스 벗어났을 때 잘못된 형식이면 빨간색 에러 표시
4. **버튼 활성화**: 유효한 전화번호 입력 시에만 버튼 활성화
   - 어드민: 유효한 번호만 있으면 OK
   - 일반 유저: 유효한 번호 + 매칭 기간 내
