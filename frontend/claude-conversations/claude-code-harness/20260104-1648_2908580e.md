# 수학 문제 풀이 스킬

## Purpose

수학 문제 이미지를 분석하여 **여러 가지 풀이 방법**을 단계별로 제공합니다.
학생들이 문제를 깊이 이해하고, 다양한 접근법을 배울 수 있도록 돕습니다.

## When to Use This Skill

- 수학 문제 이미지를 입력받았을 때
- 문제 풀이법을 요청받았을 때
- Step-by-step 설명이 필요할 때
- 여러 풀이 방법 탐색이 필요할 때
- 수학 개념 학습이 필요할 때

---

## Core Features

### 1. 다중 풀이법 제시
- 최소 2-3가지 서로 다른 접근법 제공
- 표준 방법 + 직관적 방법 + 고급 기법
- 각 방법의 난이도 표시 (⭐~⭐⭐⭐)
- 장단점 비교 설명

### 2. 웹 검색 통합
- 풀이법이 불확실할 경우 자동 검색
- web-research-specialist 에이전트 활용
- Khan Academy, Wolfram MathWorld 등 신뢰할 수 있는 소스
- 여러 소스의 일치하는 내용 확인

### 3. 학생 친화적 설명
- 각 단계의 목적 명시
- "왜 이렇게 하는가?" 설명 포함
- 시각화 활용 (그래프, 표, 다이어그램)
- 친근한 언어 사용

### 4. 구조화된 Markdown 출력
- 명확한 섹션 구분
- 이모지로 가독성 향상
- 수식 지원 ($...$)
- 파일로 저장하여 참고 용이

## Mathematical Domains

### 대수 (Algebra)
- 방정식: 일차, 이차, 고차, 연립
- 부등식: 동치 변형, 경우 나누기
- 함수: 그래프, 성질, 역함수, 합성함수

### 기하 (Geometry)
- 평면기하: 합동, 닮음, 삼각비
- 좌표기하: 직선, 원, 포물선, 타원
- 벡터: 성분, 내적, 외적

### 해석 (Calculus)
- 극한: 로피탈, 샌드위치 정리
- 미분: 정의, 공식, 응용 (접선, 극값)
- 적분: 치환, 부분, 넓이, 부피

### 확률/통계
- 경우의 수: 순열, 조합
- 확률: 조건부, 베이즈 정리
- 통계: 평균, 분산, 정규분포

각 영역의 상세 패턴은 **[resources/solution-patterns.md](resources/solution-patterns.md)** 참조

## Problem Type Classification

문제 유형 분류 체계 (3단계):

**레벨 1: 영역**
- 대수 (Algebra)
- 기하 (Geometry)
- 해석 (Analysis/Calculus)
- 확률/통계 (Probability/Statistics)

**레벨 2: 주제**
- 예: 대수 → 방정식, 부등식, 함수, 수열

**레벨 3: 세부 유형**
- 예: 방정식 → 일차방정식, 이차방정식, 고차방정식

상세 분류 기준은 **[resources/problem-types.md](resources/problem-types.md)** 참조

## Presentation Guidelines

학생 친화적인 풀이 작성 원칙:

### 1. 명확성 (Clarity)
- 각 단계의 목적 명시
- "왜 이렇게 하는가?" 설명
- 수식과 설명의 균형

### 2. 점진성 (Progression)
- 쉬운 것부터 어려운 것으로
- 각 단계가 다음 단계의 기반
- 논리적 연결 명확히

### 3. 시각화 (Visualization)
- 그래프, 표, 다이어그램 활용
- 복잡한 수식은 단계별로 분해
- Markdown 지원 범위 내에서 강조

### 4. 다양성 (Diversity)
- 여러 풀이법 제시
- 각 방법의 장단점 설명
- 학생이 선택할 수 있도록

### 5. 완결성 (Completeness)
- 답 검증 과정 포함
- 실수하기 쉬운 부분 경고
- 관련 개념/문제 연결

상세 가이드는 **[resources/presentation-guide.md](resources/presentation-guide.md)** 참조
