## 3. 레이아웃 시스템

### 페이지 사이즈
- 크기: A5 (148mm x 210mm)
- 여백: 상 20mm, 하 25mm, 좌우 18mm
- 본문 영역: 112mm x 165mm

### 그리드 시스템
```
┌─────────────────────────────┐
│         상단 여백 20mm        │
├───┬───────────────────┬───┤
│   │                   │   │
│18 │    본문 영역       │18 │
│mm │   112 x 165mm     │mm │
│   │                   │   │
├───┴───────────────────┴───┤
│     하단 여백 25mm (페이지 번호) │
└─────────────────────────────┘
```

### 챕터 시작 페이지
- 챕터 번호: 우측 상단, #FFB6C1 원형 배경
- 챕터 제목: 중앙 정렬, 상단에서 40% 위치
- 부제목: 제목 아래 8mm
- 장식선: #D4A5A5, 0.5pt

│                                 │
└─────────────────────────────────┘

테두리: #FFD1DC, 1pt
좌측 강조선: #FFB6C1, 3pt
```

### 케이스 스터디 박스
```
┌─────────────────────────────────┐
│ 📍 CASE STUDY: aimelbie         │
│ ═══════════════════════════════ │
│                                 │
│ 케이스 내용                       │
│                                 │
└─────────────────────────────────┘
배경: #FFFAFA
테두리: #D4A5A5, 1.5pt
```

```

     인용 내용이 들어갑니다.

                           ❞
                    — 출처
```
폰트: Italic, #666666
장식: #FFB6C1

### 체크리스트

```

텍스트: #2D2D2D
