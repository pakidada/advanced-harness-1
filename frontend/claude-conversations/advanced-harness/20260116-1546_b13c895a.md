## 🚨 규정 위반 없이도 밴당하는 주요 원인

### 1. 봇으로 오인되는 활동 패턴
- 시간당 50개 이상 팔로우/좋아요
- 급격한 활동 증가 (갑자기 많은 활동)
- 짧은 시간 내 과도한 댓글

### 2. 금지된 해시태그 사용
- `#beautyblogger`, `#besties` 같은 무해해 보이는 해시태그도 금지 목록에 있음
- **확인 방법**: 인스타그램에서 해시태그 검색 → "Recent posts are hidden" 경고 뜨면 금지된 것

### 3. 대량 신고 (악의적 신고)
- 동일 카테고리로 반복 신고되면 자동 제한 가능
- 경쟁자나 악성 사용자의 대량 신고로 피해 발생

### 4. 제3자 앱 사용
- 팔로워 늘리기 앱, 자동화 도구, Chrome 확장 프로그램 등

---

## 🛡️ 예방 수칙

1. **해시태그 사전 확인** - 사용 전 금지 여부 체크
2. **자연스러운 속도 유지** - 급격한 활동 증가 피하기
3. **공식 앱만 사용** - 제3자 자동화 도구 금지
4. **2단계 인증 활성화** - 계정 보안 강화
5. **트롤과 상호작용 금지** - 악성 사용자는 차단/신고

## 🔍 DM 밴의 가능한 원인 (이 중 해당되는 게 있나요?)

### 1. 발송 개수 초과
| 계정 나이 | 안전한 일일 한도 | 시간당 한도 |
|----------|-----------------|------------|
| 30일 미만 | 20-50개 | 2-3개 |
| 30-180일 | 80-120개 | 5-7개 |
| 180일 이상 | 100-200개 | 8-10개 |

### 2. 동일/유사 메시지 반복
- 같은 내용을 여러 사람에게 복붙해서 보내면 **봇으로 간주**
- 템플릿 메시지도 위험

### 3. 발송 속도 문제
- **5분 내 30개 DM** → 매우 위험
- **하루 종일 60개 분산** → 상대적으로 안전
- 짧은 시간에 몰아서 보내는 게 문제

### 4. 비팔로워에게 대량 메시지
- 팔로우 안 하는 사람에게는 **1번만** 메시지 가능 (수락 전까지)
- 신규 계정은 시간당 1-2개, 하루 5-10개만 가능

### 5. 링크 포함
- 첫 메시지에 링크 포함 시 스팸으로 판단
- 같은 링크 반복 발송도 위험

## ❌ 거절하는 이유

### 1. Instagram 서비스 약관 위반
- 비공식 자동화 도구 사용은 **명시적으로 금지**되어 있습니다
- 앞서 조사에서도 나왔듯이, 자동화 도구 사용이 **밴의 주요 원인** 중 하나입니다

### 2. 역효과 가능성
- 자동화 코드를 만들어드려도 **더 빨리 밴당할 가능성**이 높습니다
- Instagram은 브라우저 자동화, API 스크래핑 등을 적극적으로 탐지합니다

### 3. 계정 보안 위험
- 비공식 방법으로 로그인하면 계정이 **영구 정지**될 수 있습니다
- 크리덴셜 유출 위험도 있습니다

## ✅ 대신 제안드리는 방법

### 합법적인 DM 자동화 도구 (Instagram 공식 API 사용)
비즈니스 계정이라면 다음 도구들이 Instagram 정책을 준수합니다:

| 도구 | 용도 |
|-----|------|
| **ManyChat** | 챗봇, 자동 응답 |

1. **템플릿 + 개인화**: 기본 템플릿을 만들되 각 메시지마다 이름/프로필 언급
2. **시간 분산**: 알람 설정해서 하루 종일 조금씩 보내기
3. **스프레드시트 관리**: 보낸 사람, 응답 여부 추적
