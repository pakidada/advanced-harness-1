# 예비창업패키지 사업계획서 작성 노하우 & 팁

> 실전 작성 팁 중심 가이드 | 2024-2025년 기준

---

## 🎯 핵심 요약: 합격의 3가지 조건

1. **구체성**: 모든 계획은 숫자와 날짜로
2. **현실성**: 6개월 내 실행 가능한 계획
3. **실행력**: 이미 시작한 것들을 보여주기 (프로토타입, 베타 고객, 시장 검증)

---

## 📐 PSST 프레임워크 (필수 구조)

사업계획서는 이 4가지가 **유기적으로 연결**되어야 합니다:

```
Problem (문제) → Solution (해결책) → Scale-up (확장) → Team (팀)
```

**작성 프로세스**:
1. 1차 초안: 전체 흐름 가볍게
2. 2~3차 수정: 세부 내용 보완
3. 검증: PSST 논리 연결성 확인

---

## 🎓 합격의 "한 끗" - 실행력 증명

### 심사위원이 보고 싶어하는 증거

✅ **앱/프로토타입 완성**
- "돈만 주면 바로 프로덕트화 가능" 상태
- 베타 버전 제작 완료 + 스크린샷 첨부

✅ **시장 검증 완료**
- 설문/인터뷰 결과 데이터
- 베타 테스터 피드백
- 실제 사용자 리텐션 수치

✅ **확보된 고객**
- MOU 체결 업체 (서명 받기)
- Cold DM으로 확보한 잠재 고객 수
- 사전 예약/대기자 명단

✅ **제조/개발 협의 완료**
- ODM/OEM 업체 미팅 완료
- 견적서 받아두기 (예상 단가 제시)
- 시제품 샘플 확보

💡 **핵심**: AI로 계획서 퀄리티는 높아졌지만, **알맹이(실제 준비 데이터)**가 없으면 탈락합니다.

## ⚠️ 가점 항목 (각 1점)

- 팀 창업
- 관련 특허권/실용신안권 보유
- 최근 2년 창업경진대회 수상
- 고용위기지역 거주
- 감염병 관련 기술 창업

## 🎨 도식화가 생명이다

### 초등학생도 이해할 수 있게

**❌ 긴 문장으로 설명**
```
"우리 서비스는 A와 B를 연결하여 C의 문제를 해결하고
D 시장에서 E의 방식으로 수익을 창출합니다..."
```

**✅ 도식으로 한눈에**
```
[고객] → [우리 플랫폼] → [공급업체]
   ↓          ↓            ↓
 편리함    수수료 10%    판로 확보
```

### 필수 도식화 항목

1. **BM(비즈니스 모델) 다이어그램**
   - 고객 → 플랫폼 → 공급자 흐름
   - 돈의 흐름 화살표로 표시
   - 수익 포인트 명확히

2. **GTM(시장 진입 전략) 로드맵**
   ```
   1개월: 베타 출시 → 100명
   2개월: 개선 → 500명
   3개월: 마케팅 → 2,000명
   ```

3. **자금 운용 계획**
   ```
   [1억원 지원금]
   ├─ 개발비 40% → 앱 고도화
   ├─ 마케팅 30% → 인스타/유튜브 광고
   └─ 운영비 30% → 인건비/서버비
   ```

💡 **팁**: 복잡한 설명 → 간단한 도식 하나가 10배 효과

## ❌ 5대 치명적 실수 & ✅ 해결책

### 1️⃣ 아이템만 강조하고 "나"는 없다

**❌ 실패하는 작성**:
```
"혁신적인 아이디어입니다"
"기존에 없던 서비스입니다"
```

**✅ 합격하는 작성**:
```
"5년간 이 업계에서 근무하며 고객 문제를 직접 경험"
"관련 기술 특허 보유 + 업계 네트워크 50명 확보"
```

💡 **팁**: 아이템보다 **"왜 내가 이걸 성공시킬 수밖에 없는지"**를 증명하세요.

---

### 2️⃣ 추상적인 타겟과 허술한 시장 조사

**❌ 실패하는 작성**:
```
타겟: "20~30대 여성", "모든 자영업자"
시장: "10조원 규모"
```

**✅ 합격하는 작성**:
```
타겟: "서울 강남권, 월 소득 400만원+, 헬스케어 관심 30대 직장여성"
시장: "업계 1위 A사 연 500억, 2위 B사 300억 → 성장률 15%"
```

💡 **팁**: 전체 시장 말고 **업계 1~2위 기업 실적**을 분석하세요.

---

### 3️⃣ "올해 안에 런칭"같은 모호한 계획

**❌ 실패하는 작성**:
```
"올해 안에 앱 런칭"
"점진적으로 사용자 확보"
```

**✅ 합격하는 작성**:
```
1개월: 베타 100명 → 피드백 수집
2개월: 개선 → 정식 출시
3개월: 인스타 광고 → 가입 1,000명
4개월: 첫 매출 500만원
```

💡 **팁**: 6개월 안에 달성할 **월별 액션 + KPI 숫자**를 명시하세요.

---

### 4️⃣ 비현실적인 예산 (1억 넘게 쓰기)

**❌ 실패하는 작성**:
```
총 사업비: 1.5억원
신청액: 1억원
```

**✅ 합격하는 작성**:
```
총 사업비: 8,000만원
- 개발비: 4,000만원
- 마케팅비: 2,000만원
- 운영비: 2,000만원
```

💡 **팁**: 최대 1억원, 현실적인 금액으로 책정하세요.

---

### 5️⃣ PSST가 따로 놀기

**❌ 문제점**:
- 문제(P) → 해결책(S) → 확장(S) → 팀(T)이 끊김
- 각 파트가 독립적으로 작성됨

**✅ 해결책**:
- 각 파트 작성 시 이전 파트를 참조
- "위에서 언급한 ~" 식으로 연결

## 📝 섹션별 작성 노하우

### 📊 시장/아이템 분석 (⭐⭐⭐⭐⭐)

**핵심**: 시장 분석 > 아이템 설명

**작성 팁**:
```
❌ "배달 시장 20조원"
✅ "배달의민족 15조, 쿠팡이츠 5조 → 틈새시장 '건강식 배달' 30% 성장 중"
```

**필수 포함**:
- 업계 선두 기업 실적 분석
- 시장 검증 데이터 (설문, 인터뷰, 테스트)

---

### 📢 마케팅 전략 (⭐⭐⭐⭐)

**핵심**: 구체적 액션 + 예산 + 예상 결과

**작성 팁**:
```
❌ "인스타그램 인플루언서 마케팅"
✅ "팔로워 3~5만 인플루언서 10명 컨택 → 5명 답변
    건당 50만원 × 5명 = 250만원
    예상: 도달 3만명 × 전환 2% = 600명 유입"
```

---

### 👥 팀 구성 (⭐⭐⭐⭐⭐)

**핵심**: 팀 역량 = 시장 진입 전략 + 커리어 연결성

**작성 팁**:
```
대표 (본인):
- 업계 5년 경력 → 고객 pain point 직접 경험
- 네트워크 50명 → 초기 고객 확보 가능
- 이 사업과의 연결: "3년간 A사에서 유사 제품 마케팅 담당,
  고객 불만 1,000건 분석 → 이 사업 아이템 도출"

CTO (확정):
- 관련 기술 특허 보유 → 기술 경쟁 우위
- 앱 개발 3년 → 베타 버전 이미 제작 완료

마케터 (예정):
- 인스타그램 10만 팔로워 보유
- 광고 대행 2년 → 월 500만원 예산으로 ROAS 300% 달성 경험

영업 (외주):
- B2B 영업 전문 프리랜서 (건당 계약)
- 예상 단가: 계약당 수수료 10%
```

**💡 중요**:
- "예정"인 팀원도 넣을 수 있음 (합류 의향서 첨부)
- 외주 계획도 명시 (예산 + 예상 성과)
- **커리어와 이 사업의 연결고리 반드시 설명**

---

### 💰 수익 모델 (⭐⭐⭐⭐)

**핵심**: 단기/중기/장기 계획 + BM 도식화

**작성 팁**:
```
1~3개월: 플랫폼 수수료 10% → 거래액 1,000만원 = 100만원
4~6개월: 광고 추가 → 월 10만원 × 10개사 = 100만원
7개월~: 구독 모델 → 월 5,000원 × 500명 = 250만원
```

**BM 도식 예시**:
```
[고객] --결제--> [우리 플랫폼] --수수료 10%--> [판매자]
   ↓                  ↓                          ↓
 편리함          수익 창출                   판로 확보

수익 포인트:
1) 거래 수수료 10%
2) 프리미엄 광고 월 10만원
3) 구독료 월 5,000원
```

---

### 💵 자금 조달 계획 (⭐⭐⭐⭐⭐)

**핵심**: 지원금 후에도 계속 운영 가능함을 증명

**작성 팁**:
```
[예비창업패키지 6개월 후 자금 계획]

1차: 자기 자본 투입
- 본인 3,000만원 (저축금)
- 공동창업자 2,000만원
- 합계: 5,000만원

2차: 정책 자금 확보
- 기술보증기금: 신용보증 5,000만원
- 신용보증기금: 창업기업 특례보증 3,000만원
- 합계: 8,000만원 확보 가능

3차: 엔젤/VC 투자 유치
- 초기 매출 달성 후 시드 투자 5억원 목표
```

**💡 중요**: "돈 떨어지면 망한다"가 아니라 "계속 조달 계획 있다"를 보여주기

---

### 🎯 차별성 전략 (⭐⭐⭐⭐⭐)

**핵심**: 똑같은 아이템이 많다 → 브랜딩/마케팅이 관건

**❌ 흔한 차별성**:
```
"AI 기술 활용"
"사용자 친화적 UI/UX"
"합리적인 가격"
```

**✅ 진짜 차별성**:
```
1. 브랜딩 차별화
   - 타겟: "환경 의식 높은 MZ세대"
   - 컨셉: "지속가능한 라이프스타일 브랜드"
   - 증거: 인스타그램 계정 1만 팔로워 확보 완료

2. 마케팅 차별화
   - 인플루언서 5명과 MOU 체결 완료
   - 예상 비용: 건당 50만원 × 5명 = 250만원
   - 예상 효과: 도달 15만명, 전환 2% = 3,000명 유입

3. 공급망 차별화
   - ODM 업체 A사와 협의 완료
   - 견적: 단가 1만원 (100개 기준)
   - 경쟁사 대비 30% 저렴
```

**💡 팁**: "우리만의 차별점" 말고 **"이미 확보한 차별적 자산"** 제시

## 🗺️ 로드맵 & GTM 전략

### 6개월 실행 로드맵 (필수)

**도식으로 작성**:
```
[1개월차: 제품 완성]
- 베타 앱 최종 테스트
- ODM 업체와 첫 제품 생산 (100개)
- KPI: 베타 사용자 100명 확보

[2개월차: 시장 진입]
- 정식 출시 + 보도자료 배포
- 인플루언서 마케팅 시작
- KPI: 가입자 500명, 첫 매출 300만원

[3개월차: 성장 가속]
- 페이스북/인스타 광고 집행 (예산 100만원)
- B2B 영업 시작 (기업 고객 타겟)
- KPI: 가입자 2,000명, 매출 1,000만원

[4~6개월차: 확장]
- 추가 기능 개발 (피드백 반영)
- 리텐션 개선 활동
- KPI: 가입자 5,000명, 월 매출 2,000만원
```

### GTM (Go-to-Market) 전략

**초기 고객 확보 방법**:
```
1. Cold DM 전략
   - 타겟 고객 리스트 500명 확보 완료
   - 1차 DM 발송 → 응답률 10% 예상 = 50명
   - 전환율 50% 예상 = 25명 최초 고객

2. 커뮤니티 공략
   - 네이버 카페 "○○○" (회원 5만명) 홍보 협의
   - 클럽하우스/디스코드 채널 개설
   - 예상 유입: 200명

3. 제휴 전략
   - MOU 체결 기업 3곳 → 직원 1,000명에게 소개
   - 전환율 5% = 50명 확보
```

**💡 핵심**: "어떻게 고객을 확보할 것인가"를 구체적 액션 + 숫자로

## 🎤 발표평가 (2차) 노하우

### 발표 형식
- **시간**: 발표 5분 + 질의 10~15분
- **PPT**: 10페이지 내외 (표지/목차 제외)
- **심사위원**: 5명 (기업가, 컨설턴트, 교수 등)

### 발표 준비 팁

**서류 통과 전부터 준비 시작**
- PPT는 PSST 구조로 간결하게
- 5분에 딱 맞춰 10회 이상 리허설
- 핵심만 말하고 나머지는 질문으로 유도

**예상 질문 30개 준비**
- 수익 모델 관련 (가장 많이 물어봄)
- 경쟁 우위 / 차별화
- 팀 구성 / 역할 분담
- 실행 계획 / 타임라인

## 📋 작성 체크리스트

### 🎯 알맹이 체크 (가장 중요!)
- [ ] **앱/프로토타입 제작 완료** (스크린샷 첨부)
- [ ] **시장 검증 데이터** (설문/인터뷰 결과)
- [ ] **확보된 고객** (MOU/Cold DM 응답/대기자 수)
- [ ] **제조사 협의 완료** (ODM 견적서 확보)
- [ ] **커리어 ↔ 사업 연결** (왜 내가 이걸 하는지)

### 📊 도식화 체크
- [ ] **BM 다이어그램** (고객→플랫폼→공급자 흐름)
- [ ] **수익 구조 도식** (돈의 흐름 화살표)
- [ ] **로드맵 도표** (1~6개월 타임라인)
- [ ] **자금 운용 계획 도식** (1억원 사용처)
- [ ] **초등학생도 이해 가능한 쉬운 용어**

### 💰 자금 계획 체크
- [ ] **지원금 1억원 사용 계획** (개발/마케팅/운영)

## 💡 최종 조언: 합격의 절대 법칙

### 1️⃣ AI 시대의 차별화는 "알맹이"

**현실**:
- AI로 사업계획서 퀄리티는 모두 상향평준화됨
- 심사위원은 **"실제로 뭘 준비했나"**만 본다

**해법**:
```
❌ "혁신적인 AI 기반 플랫폼입니다"
✅ "앱 제작 완료 + MOU 3곳 + 베타 고객 100명 확보"
```

### 2️⃣ "돈만 주면 된다" 상태 만들기

**준비 상태 증명**:
- 앱/프로토타입 완성 (90% 이상)
- ODM/OEM 협의 완료 (견적서 보유)
- 초기 고객 확보 (MOU/대기자)
- 팀 구성 완료 (핵심 인력)

→ **"지원금 받으면 바로 시작할 수 있다"** 를 증명

### 3️⃣ 도식화 = 이해 속도 10배

**규칙**:
- 한 페이지에 하나의 도식
- 초등학생도 이해할 수 있는 단어
- 화살표로 흐름 표시
- 숫자는 크게 강조

### 4️⃣ 차별성은 "이미 확보한 것"

**❌ 실패하는 차별성**:
```
"AI 기술 활용 예정"
"사용자 친화적 UI 개발 예정"
```

**✅ 합격하는 차별성**:
```
"인스타그램 계정 1만 팔로워 → 브랜딩 자산 확보"
"인플루언서 5명 MOU → 마케팅 채널 확보"
"ODM A사 견적 → 경쟁사 대비 30% 저렴"
```

### 5️⃣ 커리어 스토리가 있어야 한다

**연결고리 만들기**:
```
"저는 3년간 화장품 마케팅을 했습니다.
1,000건의 고객 불만을 분석하며
'성분 투명성'에 대한 니즈를 발견했습니다.
그래서 이 사업을 시작합니다."
```

→ **"왜 이 사람이 이 사업을 하는가"**가 명확해야 함

### 6️⃣ 숫자로 말하기

```
"빠르게" ❌ → "3개월 안에" ✅
"많이" ❌ → "1,000명" ✅
"성장할 것" ❌ → "월 15% 성장" ✅
"준비 중" ❌ → "90% 완성" ✅
```
