# Jewelry Image Generator

Generate visually consistent jewelry product images with a lovely, princess-like pink aesthetic using Replicate MCP and the Google Nano Banana model.

## Purpose

Transform jewelry product concept photos into styled images featuring:
- Lovely, princess-like aesthetic
- Pink and rose gold color palette
- Elegant jewelry box presentation
- **Paired reference images â†’ 5 outputs per pair**

---

## File Naming Convention (IMPORTANT)

### Pairing System

Files are paired by naming pattern: `{number}-{sub}`

**Examples:**
```
1-1.jpg, 1-2.jpg  â†’ Pair 1 â†’ Output: result_1_1.jpg ~ result_1_5.jpg (5 images)
2-1.jpg, 2-2.jpg  â†’ Pair 2 â†’ Output: result_2_1.jpg ~ result_2_5.jpg (5 images)
3-1.jpg, 3-2.jpg  â†’ Pair 3 â†’ Output: result_3_1.jpg ~ result_3_5.jpg (5 images)
```

### How Pairing Works

| Input Files | Pair # | Output |
|-------------|--------|--------|
| `1-1.jpg` + `1-2.jpg` | 1 | **5 images** (result_1_1 ~ result_1_5) |
| `2-1.jpg` + `2-2.jpg` | 2 | **5 images** (result_2_1 ~ result_2_5) |
| `3-1.jpg` + `3-2.jpg` | 3 | **5 images** (result_3_1 ~ result_3_5) |

**Rule:** Same prefix number = same pair â†’ combined as references â†’ **5 outputs**

---

## Workflow

### Step-by-Step Process

1. **Identify Pairs from Filenames**
   ```
   Input: 1-1.jpg, 1-2.jpg, 2-1.jpg, 2-2.jpg

   Pairs:
   - Pair 1: [1-1.jpg, 1-2.jpg]
   - Pair 2: [2-1.jpg, 2-2.jpg]
   ```

2. **Compress & Upload Each Image**
   ```bash
   # Compress
   sips -Z 512 -s format jpeg -s formatOptions 50 input.jpg --out /tmp/compressed.jpg

   # Upload to litterbox (1 hour expiry)
   curl -sF "reqtype=fileupload" -F "time=1h" \
        -F "fileToUpload=@/tmp/compressed.jpg" \
        https://litterbox.catbox.moe/resources/internals/api.php
   ```

3. **Generate 5 Images Per Pair (Call API 5 times in parallel)**

   **IMPORTANT:** Nano Banana does not support `num_outputs` parameter.
   To generate 5 images per pair, call the API **5 times in parallel** with the same inputs.

   ```yaml
   # Call 1 of 5
   mcp__replicate__create_predictions:
     version: "google/nano-banana"
     input:
       prompt: "Transform: place this jewelry in pink velvet box..."
       image_input: ["url_1-1", "url_1-2"]
       aspect_ratio: "4:5"
       output_format: "jpg"

   # Call 2 of 5 (same parameters, parallel)
   mcp__replicate__create_predictions:
     version: "google/nano-banana"
     input:
       prompt: "Transform: place this jewelry in pink velvet box..."
       image_input: ["url_1-1", "url_1-2"]
       aspect_ratio: "4:5"
       output_format: "jpg"

   # ... repeat for calls 3, 4, 5
   ```

4. **Save Outputs**
   ```bash
   curl -o result_1_1.jpg "[output_url_1]"
   curl -o result_1_2.jpg "[output_url_2]"
   curl -o result_1_3.jpg "[output_url_3]"
   curl -o result_1_4.jpg "[output_url_4]"
   curl -o result_1_5.jpg "[output_url_5]"
   ```

## Replicate MCP Integration

### Google Nano Banana Model

**Model:** `google/nano-banana`

**Input Parameters:**
| Parameter | Type | Description |
|-----------|------|-------------|
| `prompt` | string | Style transformation prompt |
| `image_input` | array | URLs of paired reference images |
| `aspect_ratio` | string | `4:5`, `1:1`, `16:9`, `9:16`, etc. |
| `output_format` | string | `jpg` or `png` |

**Note:** No `num_outputs` parameter. Generate multiple images by calling API multiple times.

### MCP Call Template (5x Parallel Calls)

For each pair, make **5 parallel API calls** with identical parameters:

```yaml
# All 5 calls use identical parameters - run in parallel
mcp__replicate__create_predictions:
  version: "google/nano-banana"
  input:
    prompt: "Transform: place this jewelry in a pink velvet jewelry box, lovely princess-like aesthetic, soft pink and rose gold tones, scattered rose petals, dreamy bokeh, luxury jewelry advertisement, 8k quality"
    image_input:
      - "https://litter.catbox.moe/xxx.jpg"  # Image X-1
      - "https://litter.catbox.moe/yyy.jpg"  # Image X-2
    aspect_ratio: "4:5"
    output_format: "jpg"
  Prefer: "wait"
  jq_filter: "{id, status, output, error}"
```

**For 1 pair (5 images):**
- Make 5 parallel `mcp__replicate__create_predictions` calls
