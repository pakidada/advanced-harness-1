## 1.1 팬덤 경제의 규모와 기회

---

### 숫자로 보는 팬덤 경제

K-pop 팬덤은 단순한 음악 소비를 넘어 하나의 **거대한 경제 생태계**를 형성하고 있습니다.

```
╭─────────────────────────────────╮
│                                 │
│         $12.7B                  │
│   K-pop 글로벌 시장 규모 (2024)   │
│                                 │
╰─────────────────────────────────╯
```

**핵심 통계:**

| 지표 | 수치 | 의미 |
|------|------|------|
| 글로벌 K-pop 팬 | 1억 7천만 명 | 전 세계 2% |
| 연간 성장률 | 15%+ | 고속 성장 지속 |
| 팬 1인당 연간 지출 | $1,422 | 높은 구매력 |
| 굿즈 시장 규모 | $8.9B | 2028년 예상 |

### 팬덤 경제의 특징

**1. 감정적 투자가 금전적 투자로 전환**

팬들은 아이돌에 대한 사랑을 **소비**로 표현합니다. 이것은 단순한 구매가 아니라 **정체성의 표현**입니다.

**2. 높은 가격 민감도? 틀렸습니다**

일반적인 소비재와 달리, 팬덤 상품은 가격 민감도가 낮습니다. "내 최애가 쓴 것"이라면 가격은 부차적인 문제가 됩니다.

**3. 바이럴의 자연 발생**

팬덤 내에서 정보는 빠르게 퍼집니다. 한 명의 팬이 발견한 정보가 수십만 명에게 전달되는 데 단 몇 시간이면 충분합니다.

> "어떤 팬덤을 타겟할 것인가"를 먼저 정해야 합니다.

## 1.2 K-pop 시장 심층 분석

---

### K-pop 관련 상품 카테고리 맵

K-pop 팬들이 구매하는 상품은 크게 다음과 같이 분류됩니다:

```
┌─────────────────────────────────────────────┐
│              K-pop 관련 상품 시장              │
├─────────────┬─────────────┬─────────────────┤
│   공식 굿즈   │  팬메이드   │   셀럽 착용품    │
├─────────────┼─────────────┼─────────────────┤
│ • 앨범       │ • 포토카드   │ • 의류          │
│ • 포토북     │ • 슬로건     │ • 주얼리 ◀      │
│ • 콘서트 MD  │ • 인형      │ • 액세서리      │
│ • 시즌그리팅 │ • 스티커     │ • 화장품       │
└─────────────┴─────────────┴─────────────────┘
```

### 왜 "셀럽 착용품" 시장인가?

**공식 굿즈의 한계:**
- 소속사가 독점 → 진입 불가능
- 가격 결정권 없음
- 창의성 제한

**팬메이드의 한계:**
- 저작권 리스크
- 품질 신뢰도 이슈
- 가격 천장 존재

**셀럽 착용품의 기회:**
- ✅ 합법적 영역
- ✅ 프리미엄 가격 가능
- ✅ 브랜드 구축 가능
- ✅ 셀럽 변경에도 유연

```

## 1.4 니치 시장 발견 프레임워크

---

### NICHE 프레임워크

성공적인 니치 시장을 발견하기 위한 5단계 프레임워크입니다.

```
N ─── Need (필요)
      └─ 타겟 고객의 충족되지 않은 욕구는?

I ─── Identity (정체성)
      └─ 구매가 고객의 정체성과 어떻게 연결되는가?

C ─── Community (커뮤니티)
      └─ 타겟 고객들이 모이는 커뮤니티가 있는가?

H ─── Habit (습관)
      └─ 반복 구매로 이어질 수 있는가?

E ─── Emotion (감정)
      └─ 감정적 연결을 만들 수 있는가?
```

---

| 요소 | aimelbie 적용 사례 |

| **C**ommunity | 트위터, 인스타그램, 위버스 등 팬 커뮤니티 활성화 |
| **H**abit | 컴백, 콘서트, 시상식마다 새로운 착용 아이템 → 반복 구매 유도 |
| **E**motion | "내 최애와 똑같은 것을 가졌다"는 감정적 만족 |

## 2.1 셀럽 착용의 마케팅 파워

### 왜 셀럽 착용이 강력한가?

일반적인 인플루언서 마케팅과 "셀럽 착용"은 다릅니다.

```
┌─────────────────────────────────────────────┐
│           인플루언서 마케팅                   │
│                                             │
│  브랜드 → 💰 → 인플루언서 → 콘텐츠 → 팔로워   │
│                                             │
│  • 비용 발생                                 │
│  • "광고"로 인식됨                           │
│  • 신뢰도 제한적                             │
└─────────────────────────────────────────────┘

┌─────────────────────────────────────────────┐

aimelbie는 직접적으로 가격을 비교하지 않지만, **암묵적 앵커링**을 활용합니다:

- "셀럽이 착용한" = 프리미엄 연상

### 가격대별 심리적 임계점

```
         구매 저항
              │
              │    ╭─────╮
        높음  │    │     │
              │    │     │
              │────┤     ├────
              │    │     │
        낮음  │    │     │
              │    ╰─────╯
              └─────────────────
                $10  $30  $50

           ◀───────▶

            ($11-29)
```

**$10 이하:** 품질 의심
**$11-29:** 스위트 스팟 (충동구매 + 품질 신뢰)
**$30-50:** 고민 시작
**$50 이상:** 심각한 고민

---

### 가격 책정 공식

```
최적 가격 = (지각된 가치 × 0.3) + 원가 + 마진

예시:
• 셀럽 착용 목걸이의 지각된 가치: $100
• 원가: $5
• 목표 마진: $15

최적 가격 = ($100 × 0.3) + $5 + $15 = $50

실제 책정: $25 (추가 할인감 제공)
```
