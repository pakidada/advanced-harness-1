# KakaoTalk Psychology Responder

## Purpose

Generate psychologically-informed responses to KakaoTalk messages using multiple psychology frameworks. This skill combines:

- **Romantic Psychology** (연애 심리학) - Attraction, attachment, relationship dynamics
- **Evolutionary Psychology** (진화 심리학) - Mate selection, signaling, reproductive strategies
- **Cognitive Psychology** (인지 심리학) - Decision-making, perception, emotional processing
- **Social Psychology** (사회 심리학) - Influence, reciprocity, social norms

## Core Response Patterns

### Pattern 1: The Scarcity Effect (희소성 효과)

```
❌ 즉시 모든 것에 응답
✅ 선택적으로 일부만 응답하고 나머지는 다음에

Example:
상대: "오늘 뭐했어? 밥은 먹었어? 주말에 뭐해?"
응답: "오늘 친구 만났어! 주말은 아직 모르겠다 ㅎㅎ 너는?"
(의도적으로 '밥' 질문 생략 → 다음 대화 소재)
```

### Pattern 2: Intermittent Reinforcement (간헐적 강화)

```
❌ 매번 같은 패턴의 응답
✅ 가끔 예상을 벗어나는 응답

Example:
평소: 이모티콘 1-2개 사용
가끔: 이모티콘 없이 진지하게 or 이모티콘 폭탄
→ 예측 불가능성 = 관심 유지
```

### Pattern 3: Reciprocity with Slight Delay (호혜성 + 지연)

```
❌ 즉시 똑같이 반응
✅ 적절한 시간 후 약간 더 좋은 반응

상대: "보고싶다"
응답 (5-15분 후): "나도! 요즘 바쁘긴 한데 시간 내볼게"
→ 긍정 응답 + 가치 암시 + 구체적 행동 약속
```

### Pattern 4: Push-Pull Dynamic (밀당)

```
관심 표현 (Pull) + 약간의 거리감 (Push) 조합

Example:
"진짜 재밌다 ㅋㅋ 근데 나 이제 자야할듯!"
"너 완전 내 스타일인데, 쉽게 안 만나줄거야 ㅋㅋ"
```

## Response Timing Strategy

### Optimal Response Delays

| Situation | Recommended Delay | Psychology Basis |
|-----------|------------------|------------------|
| 첫 메시지 | 1-4시간 | 희소성 + 가치 신호 |
| 일반 대화 | 10-30분 | 관심 + 바쁜 생활 균형 |
| 감정적 메시지 | 5-15분 | 공감 + 진정성 |
| 깊은 질문 | 30분-2시간 | 진지한 고려 암시 |
| 테스트 메시지 | 무응답 or 24시간+ | 비반응 = 가치 |

## Message Type Analysis

### Type 1: Test Messages (테스트 메시지)

```
특징: 반응을 확인하려는 의도
예시: "뭐해?", "잠 안와?", "심심해"

분석 포인트:
- 시간대 (밤 10시 이후 = 관심 테스트)
- 빈도 (갑자기 많아짐 = 관심 상승)
- 내용 (구체적 ↓ = 테스트 ↑)

응답 전략:
- 너무 빠르면 ❌ (가용성 과잉)
- 너무 느리면 ❌ (관심 없음 시그널)
- 적당한 시간 + 약간 다른 주제로 전환 ✅
```

### Type 2: Emotional Messages (감정 메시지)

```
특징: 감정 공유, 위로 요청
예시: "오늘 너무 힘들었어", "기분이 안 좋아"

분석 포인트:
- 구체성 수준 (구체적 = 진짜 힘듦)
- 이모티콘 (😢 vs ㅠㅠ = 진지도 차이)
- 타이밍 (퇴근 후 = 하루 누적 스트레스)

응답 전략:
- 빠른 응답 (공감 신호)
- 해결책 제시 ❌, 감정 수용 ✅
- "무슨 일 있었어?" (구체화 유도)
```

### Type 3: Planning Messages (계획 메시지)

```
특징: 만남, 활동 관련
예시: "이번 주 시간 돼?", "토요일 뭐해?"

분석 포인트:
- 구체성 (날짜 명시 = 진지한 의도)
- 시간 여유 (급한 제안 vs 미리 계획)
- 제안 주체 (누가 먼저 제안?)

응답 전략:
- 즉답 ❌ (쉽게 얻는 느낌)
- 확인 후 연락 ✅ (가치 + 배려)
- 역제안 가능하면 좋음 (주도권 균형)
```

## Advanced Techniques

### 1. Emotional Mirroring + Amplification

```
상대 감정 미러링 + 살짝 증폭

상대: "오늘 날씨 좋다!"
❌ "응 좋네"
✅ "진짜! 이런 날 나가면 기분 완전 좋을듯 ㅎㅎ"
```

### 2. Curiosity Gap Creation

```
완전한 답변 대신 호기심 유발

상대: "주말에 뭐해?"
❌ "집에서 쉴 것 같아"
✅ "재밌는 거 하나 생겼는데 ㅎㅎ"
```

### 3. Vulnerability Escalation

```
점진적 취약성 공개 (신뢰 구축)

Level 1: 취미, 일상
Level 2: 가치관, 꿈
Level 3: 과거, 두려움
Level 4: 깊은 감정, 비밀

규칙: 상대보다 1단계만 앞서 공개
```

## Response Templates

### Template 1: Morning Message Response

```
상대: "좋은 아침!"

분석:
- 아침에 생각났다는 신호
- 관계 유지 의도

응답 옵션:

### Template 2: Complaint Response

```
상대: "일이 너무 많아... 힘들다"

분석:
- 감정적 지지 필요
- 해결책 ❌, 공감 ✅

응답 옵션:
A) 공감: "고생 많다 ㅠㅠ 너무 무리하지 마"

C) 지지: "네가 잘하고 있다는 거 알아. 조금만 힘내!"
```

### Template 3: Flirty Message Response

```
상대: "보고싶어"

분석:
- 관심 표현
- 상호 감정 확인 의도

응답 옵션:
A) 동일: "나도 보고싶어 ㅎㅎ"
B) 밀당: "갑자기? ㅋㅋ 왜 그래"
C) 행동: "그럼 이번주 시간 맞춰볼까?"
