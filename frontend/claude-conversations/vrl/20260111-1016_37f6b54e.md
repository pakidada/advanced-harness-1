## 스레드 어그로 글 작성 에이전트 생성 완료

`.claude/agents/threads-aggro-writer.md` 파일을 생성했습니다.

### 레퍼런스 분석 결과

**@dev_shibaa (만능시바) 스타일:**
- 반말 + 도발적 톤
- "답 다 줬음", "ㅋㅋ", "ㄷㄷ" 같은 인터넷 용어
- 위기감 조성 ("아직도 안 했으면 큰일")
- 구체적 숫자 (월 순익 3천만 원 등)

**@classbinu (클래스비누) 스타일:**
- 존댓말 + 전문적 톤
- 성과 중심 (매출, 사용자 수 등)
- 역설적 통찰 ("누구나 할 수 있지만 아무나 할 수 없습니다")
- 사업 인사이트 공유

### 에이전트 기능

| 기능 | 설명 |
|------|------|
| **스타일 선택** | A(공격적), B(전문적), Mix |
| **주제 카테고리** | 바이브코딩, 연봉, AI, 창업 등 |
| **어그로 요소** | 도발, 위기감, 자신감, 구체성, 반전 |
| **출력 형식** | 본문 + 해시태그 + 어그로 포인트 분석 |

### 사용 방법

```bash
# Task agent로 실행
Task: threads-aggro-writer 
"바이브 코딩으로 돈 버는 법 주제로 어그로 글 작성해줘. Style A로"
```

글을 직접 게시하지 않고 작성만 해주며, 허위 정보나 특정인 비방은 금지됩니다.
