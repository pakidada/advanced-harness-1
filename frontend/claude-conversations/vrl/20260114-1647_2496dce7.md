### 📊 형식별 변경 방법

**PowerPoint** (가장 쉬움)
- `홈` → `바꾸기` → `글꼴 바꾸기` → 전체 일괄 변경

**HWP (한글)**
- `서식` → `글꼴 바꾸기` → 일괄 변경

**Word**
- 스타일 기반 변경 또는 전체 선택 후 글꼴 변경

**PDF**
- ⚠️ 직접 변경 어려움 → **원본 파일에서 변경 후 PDF 재변환 권장**

**LaTeX**
```latex
\setmonohangulfont{D2Coding}  % 고정폭 한글 글꼴
```
