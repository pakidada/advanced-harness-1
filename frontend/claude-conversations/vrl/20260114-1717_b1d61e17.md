## AI 일러스트 일관된 스타일 유지 방법

### 1. 도구별 핵심 기능

| 도구 | 일관성 유지 방법 |
|------|-----------------|
| **Midjourney** | `--sref` (스타일 참조 코드) + `--cref` (캐릭터 참조) |
| **DALL-E 3** | Seed 번호와 Gen ID 활용 |
| **Stable Diffusion** | LoRA 학습 (완벽한 커스터마이징) |

### 2. 프롬프트 작성 4단계 구조

```
주제 + 환경 + 스타일 + 세부사항
```

예시: "선생님 캐릭터, 교실 배경, flat vector illustration style, soft pastel colors, friendly expression"

### 3. 실전 팁

**초보자 추천**: DALL-E 3 또는 Canva AI
- 사용이 쉽고 교육용 스타일 프리셋 제공

**중급자 추천**: Midjourney
- `--sref [코드번호]`로 스타일 고정
- `--cref [이미지URL]`로 캐릭터 일관성 유지

**고급자 추천**: Stable Diffusion + LoRA
- 5-10장 이미지로 나만의 스타일 학습 가능
- ComfyUI 워크플로우로 자동화

### 4. 교안용 워크플로우

1. **스타일 정의**: 색상 팔레트, 선 굵기, 캐릭터 비율 문서화
2. **마스터 이미지 생성**: 기준이 될 이미지 1-2장 먼저 생성
3. **참조 활용**: 마스터 이미지를 참조로 시리즈 생성
4. **일관성 검증**: 생성된 이미지들 나란히 비교

상세 가이드는 `claudedocs/ai-illustration-consistency-guide.md`에 저장되어 있습니다. 더 궁금한 부분 있으시면 말씀해주세요.

## 📌 핵심 요약: AI 일러스트 일관된 스타일 유지법

### 🎯 도구별 핵심 기능 (2025-2026)

| 도구 | 핵심 기능 | 특징 |
|------|----------|------|
| **Midjourney** | `--sref` + `--cref` | 14억 개 스타일 코드, 캐릭터 참조 |
| **DALL-E 3** | Seed + GPT Memory | 대화형 일관성 유지 |
| **Stable Diffusion** | LoRA + IP-Adapter | 완전한 커스터마이징 |
| **Flux** | Kontext + Style LoRA | 스타일 가이드 자동 분석 |

---

### 🔥 Midjourney (가장 쉬움)

```
프롬프트 예시:
"교실에서 가르치는 선생님 캐릭터, flat illustration --sref 12345 --cref [이미지URL] --cw 100"
```

- `--sref [숫자]`: 스타일 고정 (색상, 질감, 분위기)
- `--cref [URL]`: 캐릭터 일관성 (얼굴, 의상)
- `--sw 0~1000`: 스타일 강도 조절
- `--sref random`: 새로운 스타일 탐색

---

### 💡 DALL-E 3 (ChatGPT 활용)

1. 이미지 생성 후 "이 이미지의 시드 번호 알려줘" 요청
2. 다음 이미지 생성 시 시드 번호 포함
3. ChatGPT 커스텀 명령어에 스타일 규칙 저장

**Photo Grid 기법**: 
```
"4-panel grid showing teacher character in different poses, consistent flat illustration style"
```

---

### 🛠️ Stable Diffusion LoRA (완전 제어)

**2025 Best Practice:**
- 이미지: 10-30장 (1024x1024 이상)
- Network Dimension: 16-32
- 학습 스텝: 1000
- VRAM: 8GB 이상

**ConsisLoRA (최신 기술):**
- 콘텐츠와 스타일을 분리 학습
- 스타일 유출 방지

---

### 🎨 교안/PPT 제작 워크플로우

```
1. 스타일 정의
   └─ 색상 팔레트, 선 굵기, 캐릭터 비율 문서화

2. 마스터 이미지 생성
   └─ 기준 캐릭터 1-2장 생성 → sref/cref 코드 확보

3. 시리즈 생성
   └─ 동일 코드로 다양한 상황 이미지 생성

4. PPT 통합
   └─ Gamma, Canva 등 활용
```
