## Quick Reference

### 사주팔자 구조

```

천간(天干):   甲         乙         丙         丁      ← 하늘 기운 (외면)
지지(地支):   子         丑         寅         卯      ← 땅 기운 (내면)
```

|------|------|-----------|

## 계산 핵심 원칙

### 1. 절기(節氣) 기준
- **년주**: 입춘(立春) 기준 (2월 4일경), 입춘 전 출생 → 전년도 간지
- **월주**: 12절기로 월 경계 결정 (입춘→인월, 경칩→묘월...)
- **일주**: 율리우스적일(JD) 기반 계산
- **시주**: 2시간 단위 (자시 23-01, 축시 01-03...)

### 2. 년상기월법 (月干 계산)
년간에 따라 정월(인월)의 천간 시작점 결정:

| 년간 | 인월 천간 시작 |
|------|---------------|
| 甲/己 | 丙(병인월) |
| 乙/庚 | 戊(무인월) |
| 丙/辛 | 庚(경인월) |
| 丁/壬 | 壬(임인월) |
| 戊/癸 | 甲(갑인월) |

### 3. 일상기시법 (時干 계산)
일간에 따라 자시(子時)의 천간 시작점 결정:

| 일간 | 자시 천간 시작 |
|------|---------------|
| 甲/己 | 甲(갑자시) |
| 乙/庚 | 丙(병자시) |
| 丙/辛 | 戊(무자시) |
| 丁/壬 | 庚(경자시) |
| 戊/癸 | 壬(임자시) |

### 💎 용신(用神) 판단

**분석 결과**: 목(木)이 과다하여 **신강 사주**

| 구분 | 오행 | 역할 |
|------|------|------|
| **용신** | **金(금)** | 과다한 목을 제어, 재물운 보완 |
| 희신 | 土(토) | 금을 생하고 목을 흡수 |
| 기신 | 水(수) | 이미 강한 목을 더 생함 |

**용신이 금인 이유**: 사주에 금이 없어 균형을 맞추고, 재성(금)을 채워 재물운과 현실감각 강화

### 🌸 성격 및 특성

#### 핵심 성격
- **지적 호기심 왕성**: 정인이 3개로 학습능력 뛰어남
- **섬세한 감수성**: 丁火 일간 특유의 세심함과 직관력
- **창의적 표현력**: 식신+상관 조합으로 예술적 재능
- **독립적 사고**: 편인의 영향으로 독창적 관점

#### 대인관계
- 따뜻하고 배려심 깊음
- 깊은 관계를 선호하며 진정성 중시
- 다소 내성적이나 신뢰받으면 깊이 교류

#### 개선점
- 결정에 시간이 걸릴 수 있음
- 실천력 강화 필요 (인성 과다)
- 현실적 재무 감각 기르기

- 사회 활동, 커리어 발전에 좋은 시기

### 🩺 건강 주의점

**금(金) 부족 → 호흡기, 피부 주의**

| 주의 부위 | 관리 방법 |
|-----------|----------|
| 호흡기 | 심호흡, 규칙적 유산소 운동 |
| 피부 | 보습 관리, 자외선 차단 |
| 간(목 과다) | 과로 금지, 충분한 휴식 |

### 💝 연애/결혼 관점

**정관이 년지에 위치** (亥 - 정관)
- 어린 시절부터 이성에 대한 관심
- 안정적이고 책임감 있는 배우자 인연
- 결혼 후 가정 중시

**일지 卯(정인)**: 배우자 자리에 정인
- 지적이고 배려심 있는 배우자
- 서로 성장시키는 관계 형성
