```javascript
// 방법 1: ID로 직접 접근 (ID가 동적이면 변경될 수 있음)
document.querySelector('#uuid-e7 .cl-text').textContent

// 방법 2: 라벨 "학번" 옆의 값 찾기 (더 안정적)
const labels = document.querySelectorAll('.cl-output');
let studentId = '';
labels.forEach((el, i) => {
  if (el.textContent.trim() === '학번') {
    studentId = labels[i + 1]?.querySelector('.cl-text')?.textContent;
  }
});

// 방법 3: 콤보박스에서 추출 (정규식)
```
