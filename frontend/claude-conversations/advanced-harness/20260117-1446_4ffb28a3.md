## Style Specifications

### Princess-Like Aesthetic

**Visual Characteristics:**
- Soft, dreamy atmosphere
- Delicate and feminine presentation
- Romantic and luxurious feeling
- Gentle, diffused lighting
- Elegant and refined composition

**Mood Keywords:**
```
lovely, princess-like, dreamy, romantic, delicate,
feminine, luxurious, elegant, refined, soft
```

### Pink Color Palette

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| Blush Pink | `#FFB6C1` | Primary background tone |
| Rose Pink | `#FF69B4` | Accent highlights |
| Dusty Rose | `#DCAE96` | Warm undertones |
| Rose Gold | `#B76E79` | Metallic accents |
| Pearl Pink | `#F8E8EE` | Soft highlights |
| Champagne | `#F7E7CE` | Neutral balance |

**Color Keywords:**
```
pink, rose, blush, dusty rose, rose gold, champagne,
soft pink tones, romantic colors, feminine palette
```

### Jewelry Box Presentation

**Setting Elements:**
- Pink velvet jewelry box (open position)
- Soft fabric lining (velvet or silk)
- Elegant presentation cushion
- Clean, uncluttered background
- Optional: scattered rose petals or pearls

**Composition Guidelines:**
- Product centered or slightly off-center
- Jewelry box at 30-45 degree angle
- Soft shadows for depth
- Bokeh background (subtle)

## Consistency Guidelines

### For Visual Consistency Across Images

1. **Use Fixed Style Keywords**
   Always include these in every prompt:
   - "pink velvet jewelry box"
   - "princess-like aesthetic"
   - "soft feminine lighting"
   - "rose gold tones"
   - "luxurious presentation"

2. **Seed Management**
   - Use same seed for related images
   - Document seeds used for future reference
   - Vary seed only when intentionally exploring options

3. **Consistent Parameters**
   ```yaml
   guidance_scale: 7.5  # Keep consistent
   num_inference_steps: 40  # Minimum for quality
   scheduler: "DPMSolverMultistepScheduler"
   ```

4. **Image Reference (img2img)**
   When user provides concept photo:
   - Use as reference image if supported
   - Maintain product shape/features
   - Apply only style transformation
   - Strength: 0.4-0.6 for balanced transformation

### Quality Checklist

Before finalizing generated images:
- [ ] Pink/rose tones visible
- [ ] Jewelry box setting present
- [ ] Soft, feminine lighting
- [ ] No harsh shadows
- [ ] Product clearly visible
- [ ] High resolution (no artifacts)
- [ ] Princess-like aesthetic achieved

## Workflow

### Step-by-Step Process

1. **Receive Concept Photo**
   - User uploads jewelry product image
   - Identify jewelry type (ring, necklace, earrings, bracelet)

2. **Construct Prompt**
   - Use base template
   - Add specific jewelry description
   - Include all style keywords
   - Add negative prompt

3. **Generate with Replicate MCP**
   ```
   Use mcp__replicate__run_model with:
   - model: "jbilcke-hf/nano-banana"
   - prompt: [constructed prompt]
   - negative_prompt: [standard negative]
   - parameters: [consistent settings]
   ```

4. **Review Output**
   - Check against quality checklist
   - Regenerate if needed with adjusted prompt
   - Document successful parameters

5. **Deliver Results**
   - Provide generated image
   - Include prompt used (for reproducibility)
   - Note any adjustments made
