### 🚀 숨겨진 보석들

**1. Shortify** - 가장 심플
```
YouTube URL 입력 → 숏폼 영상 자동 생성
```
- 설정 거의 없음
- URL 하나로 끝

**2. ViMax** - 가장 혁신적
```
텍스트 입력 → AI 감독+작가+프로듀서가 협업 → 완성 영상
```
- RAG 기반 스크립트 엔진
- 멀티 카메라 시뮬레이션
- GitHub 트렌딩 중 🔥

**3. LTX-2** (2026년 1월 출시)
```
텍스트 → 20초 4K 영상 + 오디오 자동 생성
```
- 오픈소스 중 최초 오디오-비디오 동시 생성
- 중급 GPU에서 돌아감 (RTX A6000)
- 경쟁사 대비 5-10배 빠름

**4. Wan2GP** - GPU 없어도 OK
```
8GB VRAM으로 고품질 영상 생성
```
- 저사양 친화적
- LTX 2 distilled 모델 지원
