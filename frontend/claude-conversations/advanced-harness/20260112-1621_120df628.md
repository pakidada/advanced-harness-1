# Reel Creator Skill

## Purpose

Create engaging, aggro-style short-form video content (Instagram Reels, TikTok) using Remotion with:
- Ultra fast-paced voiceovers (1.3x speed) - Z세대 속도감
- 자연스러운 구어체 (억지 유행어 금지)
- 빠른 세그먼트 (총 6.5초, 완시청률 극대화)
- Snappy 애니메이션 (spring damping: 12+, stiffness: 600+)
- Sound effects (boom, click, ding, whoosh)
- Portrait 9:16 format optimization
- Large, readable text for mobile viewing

## When to Use

- Creating Instagram Reels or TikTok videos
- Making promotional short-form content
- Building aggro/hook-style marketing videos
- Korean "어그로" style content creation

## Composition Pattern

### 1. Config File (config.ts)

```typescript
// Timing configuration - ULTRA FAST PACING for 2025-2026 aggro style
// Z세대는 3초 안에 스킵 결정 → 더 빠르게!
export const SEGMENTS = {
  hook: { start: 0, duration: 1.0 },       // 첫 1초가 승부 (시선 강탈)
  problem: { start: 1.0, duration: 1.0 },  // 고통점 즉시 자극
  solution: { start: 2.0, duration: 1.2 }, // 해결책 제시
  proof: { start: 3.2, duration: 2.0 },    // 증거/결과물 (핵심)
  cta: { start: 5.2, duration: 1.3 },      // 행동 유도
} as const;

export const TOTAL_DURATION_SECONDS = 6.5;  // 7초 미만이 최적 (완시청률 ↑)
```

### 2. Composition File (Composition.tsx)

```typescript
import {
  AbsoluteFill, Sequence, useCurrentFrame, useVideoConfig,
  Img, OffthreadVideo, staticFile, interpolate, spring, Audio,
} from "remotion";
import { createComposition } from "../../utils/createComposition";
import { secondsToFrames } from "../../config";
import { SEGMENTS, TOTAL_DURATION_SECONDS } from "./config";

// Each segment as separate component
const HookSegment: React.FC = () => {
  const frame = useCurrentFrame();
  const { fps } = useVideoConfig();

  // Spring animation for pop-in effect
  const scale = spring({
    frame,
    fps,
    config: { damping: 8, stiffness: 400, mass: 0.3 },
  });

  return (
    <AbsoluteFill className="bg-gradient-to-br from-red-600 via-orange-500 to-yellow-400 flex items-center justify-center">
      <Audio src={staticFile("audio/voiceover-{name}/hook.mp3")} volume={1} />
      <Audio src={staticFile("audio/sfx-{name}/boom.mp3")} volume={0.5} />

      <div style={{ transform: `scale(${scale})` }} className="text-center px-8">
        <div className="text-9xl mb-8">🎯</div>
        <div className="text-white text-7xl font-black leading-tight">
          짧은 훅
          <br />
          <span className="text-yellow-300 text-9xl">핵심!</span>
        </div>
      </div>
    </AbsoluteFill>
  );
};

// Main Composition assembles segments
const MyVideoComposition: React.FC = () => {
  return (
    <AbsoluteFill>
      <Sequence
        from={secondsToFrames(SEGMENTS.hook.start)}
        durationInFrames={secondsToFrames(SEGMENTS.hook.duration)}
      >
        <HookSegment />
      </Sequence>
      {/* ... more segments */}
    </AbsoluteFill>
  );
};

// Export with preset
export const MyVideoReel = createComposition({
  name: "MyVideo-Portrait",
  component: MyVideoComposition,
  durationInSeconds: TOTAL_DURATION_SECONDS,
  preset: "Portrait-1080p",  // 1080x1920 @ 60fps
});
```

## Text Size Guidelines (CRITICAL - 무조건 크게!)

Portrait 9:16 reels는 모바일에서 봄. **글자가 화면을 꽉 채워야 함.**

### 기본 원칙

1. **최소 text-7xl** - 이보다 작으면 안 보임
2. **강조는 무조건 text-9xl** - 화면에서 튀어야 함
3. **한 화면에 텍스트 2-3줄 MAX** - 많으면 읽기 힘듦
4. **여백 충분히** - px-8 이상, 꽉 차 보이지 않게

### Tailwind Size Reference

| Class | Size | Use For |
|-------|------|---------|
| `text-9xl` | 8rem (128px) | 강조 단어, 핵심 메시지 (기본!) |
| `text-8xl` | 6rem (96px) | 메인 타이틀 |
| `text-7xl` | 4.5rem (72px) | 보조 텍스트 (최소 사이즈!) |

**⛔ text-6xl 이하는 사용 금지** - 모바일에서 안 보임

## Hook Script Generation

릴스 성공의 핵심은 첫 1초의 Hook입니다. 3초 안에 스킵 결정되므로 빠르고 자연스럽게.

### 핵심 원칙

1. **자연스럽게** - 억지로 유행어 끼워넣지 않기
2. **짧고 펀치감 있게** - 한 문장에 핵심만
3. **딱딱하지 않게** - 뉴스 앵커처럼 말하지 않기
4. **대화하듯** - 친구한테 말하는 느낌

### 피해야 할 표현 (늙어 보임)

- "야 이거 봐봐" (40대 느낌)
- "실화임?", "레알?" (2010년대)
- "대박사건" (아재)
- "꿀팁 대방출" (블로그체)
- "~하는 법 알려드릴게요" (존댓말 + 유튜브 썸네일)

## 강력한 Hook 공식 6가지

심리학적 요소를 활용한 검증된 공식들. 상황에 맞게 골라서 사용.

### 공식 1: 고정관념 뒤집기

사람들이 당연하다고 생각하는 것을 뒤집어서 충격을 주는 방법.
**"내가 알고 있던 게 틀렸구나"** 라는 깨달음 유도.

**예시**:
- "맛있으면 장사 잘될 거라고? 현실은 다릅니다"
- "운동 많이 하면 살 빠진다고? 사실은 다릅니다"
- "비싼 게 좋다고? 사실은 아닙니다"
- "노력한다고 성공? 사실 90%는 실패합니다"

### 공식 2: 구체적 숫자로 충격 주기

막연한 표현보다 구체적인 숫자. 손실이나 이익과 관련된 숫자는 강력한 임팩트.

**예시**:
- "이 실수 하나로 월 500만원 매출 날릴 수도"
- "하루 10분 루틴으로 체지방 3kg 감량"
- "3개월 만에 1만 팔로워, 핵심은 3가지"
- "건축 예산 30% 절약하는 핵심 전략"

### 공식 3: 자존심 건드리기

사람들의 자존심을 살짝 건드려서 **"나는 그런 실수 안 해"** 라고 생각하게 만든 후 관심 유도.

**예시**:
- "아직도 이렇게 한다고? 그러다 진짜 큰일 납니다"
- "열심히 하는데 결과 없다면 100% 이것 때문"
- "이거 모르면서 전문가 행세?"
- "비싸게 샀다고? 진짜 효과 아세요?"

### 공식 4: 위험과 손실 경고하기

사람들은 이익보다 손실을 더 무서워함. **"이거 안 하면 큰일 날 수 있어"** 라는 위기감 조성.

**예시**:
- "이거 잘못하면 수리비만 수백만 원"
- "이 실수 하면 절대 단골 안 생깁니다"
- "제대로 안 하면 오히려 망가집니다"
- "이거 모르면 시간만 낭비입니다"

### 공식 5: 전문가 권위 빌리기

유명한 사람이나 전문가 언급으로 신뢰도 상승. **"이 정도 사람이 말하면 믿을 만하겠다"**.

**예시**:
- "10년 차 전문가가 말하는 핵심 포인트"
- "성공한 CEO들이 실천하는 아침 루틴"
- "업계 1위가 숨기는 비결"
- "연예인들도 몰래 다녀가는 곳"

### 공식 6: 비밀스러운 정보 암시하기

남들이 모르는 특별한 정보. **"이거 알면 나만 유리해질 수 있겠다"** 라는 생각 유도.

**예시**:
- "이 작은 변화 하나로 결과가 2배"
- "알려주지 않는 진짜 방법"
- "이거 하나만 바꿨는데 확 달라졌습니다"
- "99%가 모르는 핵심"
