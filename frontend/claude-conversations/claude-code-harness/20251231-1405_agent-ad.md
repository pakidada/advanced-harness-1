# 수능(CSAT) 영어 어법 구문분석 리서치 결과

## 1. 수능 어법 출제 경향 (2024-2025)

### 주요 특징
- **출제 문항**: 29번 어법 문제 1문항 (3점)
- **난이도**: 변동성이 큼 - 쉬울 때는 매우 쉽지만, 어려울 때는 빈칸 추론 수준
- **출제 방식**: 5개 밑줄 친 부분 중 어법상 틀린 부분 찾기

### 최근 트렌드 (2024-2025)
1. **해석 중심 어법**: 기술적 공식으로만 풀 수 없고, 반드시 '해석'을 동반해야 정확하게 풀림
2. **문맥 의존성 증가**: 전체 문맥과 내용을 정확히 이해해야 함
3. **종합적 사고력 요구**: 키워드 중심이나 부분 읽기로는 해결 불가

### 핵심 출제 포인트 (빈출 55개 패턴)

**동사 관련 (약 50%)**:
- 주어-동사 수 일치 (2025학년도 출제)
- 동사 형태: 원형/to부정사/-ing/과거분사
- 시제, 조동사, 수동태
- 준동사 vs 일반동사 구분

**관계사 및 접속사 (약 36%)**:
- 관계대명사 vs 관계부사
- 전치사 + 관계대명사 (2025년 5월 모의고사 출제)
- which/where/that 선택

**기타**:
- 능동/수동 구분
- 대명사 일치
- 형용사/부사 구분

출처: [영어 29번 어법 문제의 모든 것](https://orbi.kr/00073016508), [영어 문법 문제 잘 푸는 방법](https://gongbucoach.com/english-grammar-solving-tips/), [2025학년도 수능 영어 29번](https://peshare.com/sat/2025/e/29.jsp)

## 2. 구문분석 표기 규칙

### 2.1 한국 영어교육 기본 체계

**문장 성분 표기**:
- **S (Subject/주어)**: 서술어가 나타내는 동작/상태의 주체
- **V (Verb/동사)**: 주어의 내용을 전개하는 서술어
- **O (Object/목적어)**: 동사 행위의 대상
- **C (Complement/보어)**: 주어/목적어를 보충 설명

**5형식 체계** (영어 교육 근간):
1. S + V
2. S + V + C
3. S + V + O
4. S + V + IO + DO (간접목적어 + 직접목적어)
5. S + V + O + C

출처: [영어 문장 구조](https://blog.essayreview.co.kr/topic/language-rules/sentence-structure/), [문장의 5가지 요소](https://opentutorials.org/module/645/9253)

### 2.2 밑줄 사용법 (일반적 관행)

한국 영어교육에서 밑줄 사용 패턴:
- **1줄 밑줄**: 주어 또는 주요 성분 강조
- **2줄 밑줄**: 동사 강조 (일부 교재)
- **물결 밑줄**: 수식어구 표시

**수능 어법 문제에서의 밑줄**:
- 검토 대상 부분 표시 (5개 선택지)
- 동사, 관계사, 분사 등 어법 판단 대상

출처: [괄호의 기술](https://www.enago.co.kr/academy/mastering-the-art-of-parenthesis/amp/)

### 2.3 괄호 사용법

**소괄호 ( )**:
- 수식어구, 부연 설명 표시
- 전치사구 묶음
- 종속절 표시

**대괄호 [ ]**:
- 중첩 괄호 필요 시 바깥쪽 표시
- 인용문 내 설명 추가
- 생략된 주어/목적어 보충 표시

**중괄호 { }**:
- 여러 요소 묶음
- 선택 항목 표시

**띄어쓰기 규칙**:
- 한국어: 괄호 앞뒤 붙여쓰기
- 영어: 여는 괄호 앞, 닫는 괄호 뒤 띄어쓰기
  - 예: `packet loss rate (PLR)` ✓
  - 예: `packet loss rate(PLR)` ✗

출처: [둥근 괄호 & 대괄호](https://blog.essayreview.co.kr/topic/language-rules/parentheses/), [영문 국문 괄호 띄어쓰기](https://www.letmecompile.com/영문-국문-괄호-띄어쓰기/)

### 2.4 기타 표기 기호

**슬래시 /**:
- 동의어/유사어 병렬 표시
- 선택 항목 구분

**화살표 →**:
- 문장 성분 간 관계 표시
- 수식 관계 표시

**색상 코딩** (디지털 학습):
- 주어: 파란색
- 동사: 빨간색
- 목적어: 초록색
- 수식어: 회색 또는 기타

## 4. 영어 문장 다이어그램 (Sentence Diagramming)

### 4.1 Reed-Kellogg 시스템 (1877년 개발)

**기본 구조**:
- 수평선(baseline): 주어 | 동사 | 목적어
- 수직선( | ): 주어/동사/목적어 구분
- 사선( / ): 수식어, 전치사, 관사
- 점선( ⦙ ): 접속사

**표기 규칙**:

```
        주어    |    동사    |    목적어
      ─────────┼──────────┼─────────
         /           /          /
     수식어      부사      수식어
```

**목적어 구분**:
- 직접목적어: 수직선 |
- 주격보어/서술형용사: 역슬래시 \

**수식어 배치**:
- 주어/동사/목적어 아래 사선에 배치
- 전치사구: 사선(전치사) + 수평선(목적어)

출처: [Sentence Diagramming - Grammarly](https://www.grammarly.com/blog/sentences/sentence-diagramming/), [Reed-Kellogg diagram - Wikipedia](https://en.wikipedia.org/wiki/Reed–Kellogg_sentence_diagram), [How to Diagram - MasterClass](https://www.masterclass.com/articles/how-to-diagram-a-sentence)

### 4.2 현대 파싱 트리와의 관계

Reed-Kellogg 다이어그램은:
- **구성 성분 관계**(constituency relation) 반영
- **의존 관계**(dependency relation) 표현
- 현대 언어학의 parse tree 개념과 유사
- 전통적(traditional) 방식으로 분류

출처: [Reed-Kellogg diagram - Wikipedia](https://en.wikipedia.org/wiki/Reed–Kellogg_sentence_diagram)

### 4.3 한국 교육에서의 적용

- Reed-Kellogg 시스템을 직접 사용하는 경우는 드묾
- 대신 SVOC 기반 단순화된 표기 선호
- 색상 코딩 + 괄호 + 밑줄 조합이 주류

출처: [Learn Korean Forum](https://www.koreanclass101.com/forum/viewtopic.php?t=3028)

## 6. 실제 사용 예시

### 예시 1: 수능 어법 문제 분석

**원문**:
"patients recovering from a brain injury affecting the motor system knows"

**구문 분석**:
```
[patients (recovering from a brain injury) (affecting the motor system)] knows
   S주어        현재분사구1 수식             현재분사구2 수식           V동사
                    (복수 주어)                                    (단수 동사 - 오류!)
```

**표기**:
- 대괄호 [ ]: 주어 전체
- 소괄호 ( ): 수식어구
- 밑줄: 주요 성분
- 정답: knows → know (주어-동사 수 일치)

출처: [2025학년도 수능 29번](https://peshare.com/sat/2025/e/29.jsp)

### 예시 2: 관계사 구문

**원문**:
"the extent to which scientists need to develop relationships"

**구문 분석**:
```
the extent [to which scientists need to develop relationships]
    선행사    [전치사+관계대명사      관계절 내용              ]
```

**표기**:
- 대괄호 [ ]: 관계절
- 밑줄: 전치사+관계대명사 (핵심 구조)
- 화살표: to which → the extent

출처: [2025년 5월 모의고사](https://www.teacherville.co.kr/joepaulchoi/contents/25018.edu)

### 예시 3: Reed-Kellogg 다이어그램

**원문**: "John is in the park"

**다이어그램**:
```
      John  |  is
            |\
            |  \
            |   (in)
            |    \
            |     the park
```

출처: [Grammarly Sentence Diagramming](https://www.grammarly.com/blog/sentences/sentence-diagramming/)
