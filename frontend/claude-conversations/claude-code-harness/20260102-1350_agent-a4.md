# 사주분석(四柱分析, BaZi) 스킬 개발을 위한 종합 연구 리포트

## Executive Summary

사주팔자(四柱八字, BaZi, Four Pillars of Destiny)는 생년월일시를 기반으로 한 동양 운명학 시스템입니다. Python으로 구현 가능한 여러 오픈소스 라이브러리가 존재하며, 천간(天干) 10개와 지지(地支) 12개의 조합인 육십갑자(六十甲子) 체계를 기반으로 합니다. 본 리포트는 사주 계산 알고리즘, 해석 방법론, Python 구현 예시를 포함한 실용적 정보를 제공합니다.

## 1. 사주 기본 개념

### 1.1 사주팔자(四柱八字)의 구성요소

사주팔자는 **4개의 기둥(四柱)**과 **여덟 개의 글자(八字)**로 구성됩니다:
- **년주(年柱)**: 조상, 조부모 대표
- **월주(月柱)**: 부모 세대
- **일주(日柱)**: 본인과 배우자
- **시주(時柱)**: 자녀 세대

각 기둥은 **천간(天干) 1개 + 지지(地支) 1개**로 이루어져 총 8글자가 됩니다.

### 1.2 천간(天干) 10개

양간(陽干) 5개: **갑(甲), 병(丙), 무(戊), 경(庚), 임(壬)**  
음간(陰干) 5개: **을(乙), 정(丁), 기(己), 신(辛), 계(癸)**

영문 표기: Jia(甲), Yi(乙), Bing(丙), Ding(丁), Wu(戊), Ji(己), Geng(庚), Xin(辛), Ren(壬), Gui(癸)

### 1.3 지지(地支) 12개

양지 6개: **자(子), 인(寅), 진(辰), 오(午), 신(申), 술(戌)**  
음지 6개: **축(丑), 묘(卯), 사(巳), 미(未), 유(酉), 해(亥)**

영문/12지지: 자(Zi/鼠), 축(Chou/牛), 인(Yin/虎), 묘(Mao/兔), 진(Chen/龍), 사(Si/蛇), 오(Wu/馬), 미(Wei/羊), 신(Shen/猴), 유(You/鷄), 술(Xu/犬), 해(Hai/豬)

### 1.4 오행(五行) Five Elements

| 오행 | 천간 | 지지 | 방위 | 계절 | 속성 |
|------|------|------|------|------|------|
| 목(木, Wood) | 갑(양), 을(음) | 인, 묘 | 동방 | 봄 | 성장 |
| 화(火, Fire) | 병(양), 정(음) | 사, 오 | 남방 | 여름 | 확산 |
| 토(土, Earth) | 무(양), 기(음) | 진, 술, 축, 미 | 중앙 | 환절기 | 안정 |
| 금(金, Metal) | 경(양), 신(음) | 신, 유 | 서방 | 가을 | 수렴 |
| 수(水, Water) | 임(양), 계(음) | 해, 자 | 북방 | 겨울 | 저장 |

## 2. 사주 계산 방법

### 2.1 육십갑자(六十甲子) 계산 알고리즘

천간 10개 × 지지 12개 = 60가지 조합 (음양이 같은 것끼리만 결합)

**Python 구현 예시:**

```python
def calculate_ganji(year):
    """년도를 입력받아 육십갑자를 반환"""
    cheongan = ['갑', '을', '병', '정', '무', '기', '경', '신', '임', '계']
    jiji = ['자', '축', '인', '묘', '진', '사', '오', '미', '신', '유', '술', '해']
    
    if year > 0:
        # 기원후: 서기 4년이 갑자년
        cheongan_idx = (year - 4) % 10
        jiji_idx = (year - 4) % 12
    else:
        # 기원전
        adjusted_year = year - 3
        cheongan_idx = adjusted_year % 10
        jiji_idx = adjusted_year % 12
    
    return cheongan[cheongan_idx] + jiji[jiji_idx]

# 예시
print(calculate_ganji(2025))  # 을사년
print(calculate_ganji(1592))  # 임진년 (임진왜란)
```

### 2.2 만세력 계산 핵심 요소

#### 표준시 → 태양시 보정
- 한국 표준시: UTC+9 (동경 135° 기준)
- 서울 경도: 127.5° → 약 30분 차이
- **태양시 = 표준시 - 30분** (대략)

#### 서머타임 적용
- 1948년 ~ 1988년 서울올림픽까지 총 12번 시행
- 해당 기간 출생자는 시간 보정 필요

#### 자시(子時) 처리
- 전통: 23:00 ~ 01:00
- 현대: 00:00 ~ 01:00 또는 23:30 ~ 00:30 (유파마다 차이)

### 2.3 절기(24節氣) 기준 월주 결정

**봄철 6절기:**
- 입춘(立春, ~315°) → 1월 절기
- 우수(雨水, ~330°) → 1월 중기
- 경칩(驚蟄, ~345°) → 2월 절기
- 춘분(春分, 0°) → 2월 중기
- 청명(淸明, ~15°) → 3월 절기
- 곡우(穀雨, ~30°) → 3월 중기

**월주는 절기 기준으로 바뀜** (음력 기준 아님!)  
예: 2025년 2월 3일 23:09 입춘 → 이후부터 인월(寅月)

## 3. 사주 해석 방법

### 3.1 일간(日干) 기준 해석

**일간 = Day Master (日主)**  
사주의 주인공, 본인을 나타내는 핵심 요소. 일주의 천간을 기준으로 다른 7글자와의 관계를 분석합니다.

### 3.2 십성(十星) / 십신(十神) - Ten Gods

일간을 기준으로 다른 글자들과의 오행 관계를 10가지로 분류:

| 십성 | 한자 | 영문 | 관계 | 의미 | 성향 |
|------|------|------|------|------|------|
| **비견** | 比肩 | Friend | 일간과 음양·오행 동일 | 친구, 형제, 동료 | 자존심, 독립성 |
| **겁재** | 劫財 | Rob Wealth | 일간과 오행 같고 음양 다름 | 경쟁자, 재물 위협 | 도전, 경쟁심 |
| **식신** | 食神 | Eating God | 일간이 생하고 음양 같음 | 재능, 표현, 기회 | 온화, 여유 |
| **상관** | 傷官 | Hurting Officer | 일간이 생하고 음양 다름 | 창의성, 반항, 재능 | 비판적, 자유분방 |
| **편재** | 偏財 | Indirect Wealth | 일간이 극하고 음양 같음 | 횡재, 투자, 사업 | 활동적, 사교적 |
| **정재** | 正財 | Direct Wealth | 일간이 극하고 음양 다름 | 정규 수입, 안정적 재물 | 성실, 계획적 |
| **편관** | 偏官 | Unorthodox Officer (7살) | 일간을 극하고 음양 같음 | 권위, 압박, 도전 | 냉정, 원칙주의 |
| **정관** | 正官 | Proper Officer | 일간을 극하고 음양 다름 | 명예, 직위, 질서 | 책임감, 보수성 |
| **편인** | 偏印 | Indirect Resource (효신) | 일간을 생하고 음양 같음 | 전문성, 비주류 학문 | 독창적, 고집 |
| **정인** | 正印 | Direct Resource | 일간을 생하고 음양 다름 | 학문, 보호, 어머니 | 지혜, 인내 |

**길신(吉神)**: 식신, 정재, 정관, 정인, 비견  
**흉신(凶神)**: 상관, 편관(칠살), 겁재, 편인(효신)

### 3.3 용신(用神) 개념

**용신 = 사주에서 가장 필요한 오행**  
일간의 강약과 사주 전체의 균형을 고려하여 선정.

#### 용신 선정 5가지 방법

1. **억부용신(抑扶用神)**: 강하면 억제, 약하면 부축
   - 일간 강함 → 일간을 설기(洩氣)하거나 극하는 오행이 용신
   - 일간 약함 → 일간을 생하거나 돕는 오행이 용신

2. **조후용신(調候用神)**: 계절의 한열조습을 조절
   - 여름생(하절) → 수(水) 용신
   - 겨울생(동절) → 화(火) 용신

3. **통관용신(通關用神)**: 극심한 충돌을 중재
4. **병약용신(病藥用神)**: 사주의 문제점(병)을 치료
5. **격국용신(格局用神)**: 특수 격국에 맞는 용신

**관련 개념:**
- **희신(喜神)**: 용신을 돕는 오행
- **기신(忌神)**: 용신을 방해하는 오행
- **구신(仇神)**: 기신을 돕는 오행
- **한신(閑神)**: 용신, 기신, 희신, 구신 어느 것도 아닌 오행

### 3.4 대운(大運), 세운(歲運), 월운(月運), 일진(日運)

| 운 | 주기 | 의미 | 해석 포인트 |
|-----|------|------|-------------|
| **대운** | 10년 | 인생의 큰 흐름, 환경 변화 | 지지 중시, 30년마다 계절 변화 |
| **세운** | 1년 | 연간 운세 (모든 사주 동일 적용) | 천간 중시, 대운 틀 안에서 작용 |
| **월운** | 1개월 | 월간 운세, 실행 여부 | 세운에 종속, 실행 타이밍 |
| **일진** | 1일 | 일일 운세 | 월운에 종속, 60일 주기 순환 |

**해석 우선순위**: 대운 > 세운 > 월운 > 일진

### 3.5 지장간(支藏干)

**지지 안에 숨어있는 천간** (2~3개)

**구성:**
- **여기(餘氣)/초기(初氣)**: 앞달의 남은 기운
- **중기(中氣)**: 중간 기운 (다음 계절 또는 앞 계절)
- **정기(正氣)/본기(本氣)**: 지지의 본래 오행 기운

**월률분야 (일수 배분):**

| 지지 유형 | 지지 | 여기 | 중기 | 정기 | 합계 |
|-----------|------|------|------|------|------|

| **왕지** | 자(子), 오(午), 묘(卯), 유(酉) | 10일 | - | 20일 | 30일 |
| **고지** | 진(辰), 술(戌), 축(丑), 미(未) | 9일 | 3일 | 18일 | 30일 |

```python
print(f"기운: {yun.getStartYear()}세부터")
for daYun in yun.getDaYun():
    print(f"{daYun.getStartYear()}세~{daYun.getEndYear()}세: {daYun.getGanZhi()}")
```

## 5. 실용적 해석 패턴

### 5.1 성격 분석 패턴

**일간 오행별 기본 성격:**
- **갑목(甲木)**: 큰 나무 - 곧고 꿋꿋함, 리더십
- **을목(乙木)**: 작은 풀 - 유연하고 섬세함
- **병화(丙火)**: 태양 - 밝고 외향적, 열정적
- **정화(丁火)**: 촛불 - 차분하고 집중력 있음
- **무토(戊土)**: 산 - 묵직하고 신중함
- **기토(己土)**: 밭 - 포용력 있고 실용적
- **경금(庚金)**: 쇠 - 단호하고 결단력 있음
- **신금(辛金)**: 보석 - 섬세하고 고집 있음
- **임수(壬水)**: 큰 물 - 융통성 있고 사교적
- **계수(癸水)**: 이슬 - 조용하고 내면적

**십성별 성격:**
- 식신 많음 → 온화, 표현력 좋음
- 상관 많음 → 창의적, 자유분방, 비판적
- 정재 많음 → 성실, 계획적, 현실적
- 편재 많음 → 사교적, 활동적, 모험적
- 정관 많음 → 책임감, 명예 중시
- 편관 많음 → 강인함, 도전적

### 5.2 적성/직업 분석

| 십성 | 적합 직업 | 이유 |
|------|----------|------|
| **식신** | 요리사, 예술가, 강사, 서비스업 | 표현력, 재능 활용 |
| **상관** | 디자이너, 작가, 연구원, 프리랜서 | 창의성, 독창성 |
| **정재** | 회계사, 공무원, 은행원, 경리 | 성실성, 안정성 |
| **편재** | 사업가, 투자가, 영업직 | 활동성, 사교성 |
| **정관** | 법조인, 공무원, 관리직 | 책임감, 명예 |
| **편관** | 군인, 경찰, 의사, 운동선수 | 권위, 강인함 |
| **정인** | 교수, 연구원, 전문직 | 학문, 인내 |
| **편인** | 예술가, 종교인, 철학자 | 독창성, 전문성 |

**오행별 적성:**
- **목(木)**: 교육, 문화, 환경, 디자인
- **화(火)**: 방송, 엔터테인먼트, 에너지
- **토(土)**: 부동산, 건설, 농업, 중개
- **금(金)**: 금융, 기계, 법조, 의료
- **수(水)**: 유통, 무역, 물류, 여행

### 5.3 건강 분석

**오행별 취약 부위:**
- **목(木)** 과다/부족: 간, 담낭, 눈, 근육, 신경계
- **화(火)** 과다/부족: 심장, 소장, 혈관, 혀
- **토(土)** 과다/부족: 위장, 비장, 입, 살찌기 쉬움
- **금(金)** 과다/부족: 폐, 대장, 피부, 코, 호흡기
- **수(水)** 과다/부족: 신장, 방광, 귀, 생식기, 뼈

**십성별 건강 패턴:**
- 상관 과다: 스트레스성 질환
- 편관 과다: 외상, 수술 가능성
- 식신 과다: 비만, 당뇨 주의
- 인성 부족: 면역력 약화

### 5.4 대인관계 분석

**육친(六親) 관계:**
- **비견/겁재**: 형제자매, 친구
- **식신/상관**: 자녀 (여자), 할머니 (남자)
- **정재/편재**: 배우자 (남자), 아버지
- **정관/편관**: 자녀 (남자), 남편 (여자)
- **정인/편인**: 어머니, 스승

**합(合)과 충(沖):**
- **천간합**: 갑기(甲己)합토, 을경(乙庚)합금, 병신(丙辛)합수, 정임(丁壬)합목, 무계(戊癸)합화
- **지지합**: 자축(子丑)합토, 인해(寅亥)합목, 묘술(卯戌)합화, 진유(辰酉)합금, 사신(巳申)합수, 오미(午未)합화
- **지지충**: 자오(子午)충, 축미(丑未)충, 인신(寅申)충, 묘유(卯酉)충, 진술(辰戌)충, 사해(巳亥)충

## 6. 특수 격국(格局) 및 패턴

### 6.1 일반 격국 (八大格局)

1. **정관격(正官格)**: 정관이 월지에 투출
2. **편관격(偏官格/七殺格)**: 편관이 월지에 투출
3. **정인격(正印格)**: 정인이 월지에 투출
4. **편인격(偏印格)**: 편인이 월지에 투출
5. **식신격(食神格)**: 식신이 월지에 투출
6. **상관격(傷官格)**: 상관이 월지에 투출
7. **정재격(正財格)**: 정재가 월지에 투출
8. **편재격(偏財格)**: 편재가 월지에 투출

### 6.2 특수 격국 (從格 등)

**종격(從格)**: 일간이 극히 약하거나 한 오행이 극히 강할 때
- **종왕격(從旺格)**: 비겫이 강함
- **종재격(從財格)**: 재성이 강함
- **종살격(從殺格)**: 관살이 강함
- **종아격(從兒格)**: 식상이 강함

**희귀 패턴:**
- **자오쌍포(子午雙包)**: 자수와 오화가 동시 출현, 음양 균형
- **육을서귀(六乙鼠貴)**: 일간이 을이고 자시 출생 (극희귀)
- **순생천간(順生天干)**: 년간→월간→일간→시간 순차 상생
