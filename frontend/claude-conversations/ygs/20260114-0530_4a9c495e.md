**백엔드:**
- `utils/matching_window.py` 유틸리티 생성
- `GET /api/v1/matches/status` - 공개 엔드포인트 (윈도우 상태 확인)
- `GET /api/v1/matches/users/{user_id}` - 윈도우 검증 추가
  - 윈도우 외: 403 에러 + "매칭 확인 기간이 아닙니다" 메시지
  - `bypass_window=true` 파라미터로 관리자 우회 가능
