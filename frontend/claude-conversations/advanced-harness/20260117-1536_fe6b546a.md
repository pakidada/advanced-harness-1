## Claude Code 대화 기록 저장 위치

**저장 경로:**
```
~/.claude/
├── history.jsonl                         # 전체 세션 인덱스
└── projects/
    └── -Users-chajinhyeog-workspace-.../  # 프로젝트별 폴더
        └── [session-id].jsonl             # 각 대화 기록 (JSONL 형식)
```

**이전 대화 다시 보기:**

| 방법 | 명령어 |
|------|--------|
| 세션 목록 보기 | `claude --resume` 또는 `/resume` |
| 최근 대화 이어하기 | `claude --continue` |
| 특정 세션 재개 | `claude --resume 세션이름` |

**유용한 팁:**
- `/rename 작업이름` - 현재 세션에 이름 붙이기 (나중에 찾기 쉬움)
- 세션 선택기에서 `P` 누르면 내용 미리보기 가능
- `/` 누르면 세션 검색 가능

**직접 파일 확인하려면:**
```bash
ls ~/.claude/projects/
cat ~/.claude/history.jsonl | head
```

몇 가지 방법이 있어요:

**1. 마크다운으로 내보내기 (가장 간단)**
```bash
# 현재 세션을 마크다운으로 저장
/export conversation.md
```

**2. 수동으로 복사**
- `Cmd + Shift + C` - 마지막 응답 복사
- 또는 터미널에서 드래그해서 복사 후 노션/옵시디언에 붙여넣기

**3. JSONL을 읽기 좋게 변환**
```bash
# 현재 프로젝트의 최근 세션을 마크다운으로 변환
cat ~/.claude/projects/-Users-chajinhyeog-workspace-advanced-harness/*.jsonl | \
  jq -r 'select(.type == "human" or .type == "assistant") | 
  if .type == "human" then "## 🧑 User\n" + .text 
  else "## 🤖 Claude\n" + .text end' > ~/Desktop/claude-chat.md
```

**추천: 옵시디언이나 노션에 정리**
