# 예비창업패키지 사업계획서 작성 스킬 (v2.0)

예비창업패키지 사업계획서를 **2025-yebi-startup.docx 파일에 직접 작성**하는 전문 스킬입니다.

## 📁 스킬 구조

```
yebi-startup-writer/
├── SKILL.md                  # 스킬 메인 파일 (가이드라인, 코드 예제)
├── README.md                 # 이 파일
└── references/               # 참고 문서 (자동 참조)
    ├── yebi-knowhow.md       # 예비창업패키지 핵심 가이드 (65KB)
    ├── yebi-knowhow2.md      # 실전 작성 팁과 노하우 (16KB)
    └── 2025-yebi-startup.docx # 작성할 사업계획서 템플릿 (128KB)
```

## ✨ 주요 기능

### 1. Docx 파일 직접 작성
- ✅ 기존 docx 파일의 **서식/스타일 완벽 보존**
- ✅ 플레이스홀더만 찾아서 **텍스트 교체**
- ✅ 표, 도형, 이미지 **위치 유지**
- ❌ 레이아웃 변경 없음

### 2. PSST 프레임워크 적용
- **P**roblem (문제 정의)
- **S**olution (해결책)
- **S**cale-up (확장 계획)
- **T**eam (팀 구성)

### 3. 실행력 증명 중심
- 프로토타입 완성도 확인
- 시장 검증 데이터 요구
- 확보된 고객 검증
- 제조/개발 협의 확인

### 4. 자동 체크리스트
- 알맹이 체크 (실행력)
- 도식화 체크
- 자금 계획 체크
- 팀 구성 체크
- 차별성 체크

## 🚀 사용 방법

### 자동 활성화 키워드
- 예비창업
- 창업패키지
- 사업계획서
- PSST
- 창업 지원

### 워크플로우

```
1. 키워드 입력
   → "예비창업패키지 사업계획서 작성해줘"

2. 스킬 자동 활성화
   → 문서 구조 분석
   → 대화형 질문 시작

3. 내용 수집
   → PSST 각 섹션별 질문/답변

4. Docx 직접 작성
   → 기존 서식 유지하며 텍스트 교체
   → 표/도식 자동 생성

5. 체크리스트 검증
   → 누락 항목 확인
   → 최종 완성
```

## 💡 핵심 원칙

### 합격의 3가지 조건
1. **구체성**: 숫자와 날짜로 표현

3. **실행력**: 이미 시작한 것 증명
